[情報サービスの関連サービス] 情報サービス部門は主にどのような業務をサポートするのですか?

情報サービス室は主に中央研究院の各部門の研究業務の支援と事務業務のコンピュータ化を担当しています。

[情報サービスの関連サービス] 情報サービス局は関連提案を推進する際にどのような要素を考慮しますか?

情報サービス室では、関連プログラムの推進にあたり、ITアプリケーションの開発動向や病院のサービスニーズを考慮し、研究業務環境の改善や事務業務の効率化に役立つプログラムをタイムリーに開始していきます。

[情報サービスの関連サービス] 彼らが提供するサービスの 2 つのカテゴリは何ですか?

提供するサービス内容は大きく分けて、病院全体の共通サービスと特殊な事務サービスの2つに分かれます。病院全体の共有サービスは中央研究院の同僚向けの情報関連サービスであり、管理サービスは事務および情報担当者向けの関連サービスです。

[情報サービスの関連サービス] 病院全体の共有サービスは誰に提供されますか?

病院全体の共有サービスは中央研究院の同僚に提供されます。

[情報サービスの関連サービス] 情報サービス室の具体的なサービス項目や内容についてはどこに問い合わせればよいですか？

情報サービス局のウェブサイト https://its.sinica.edu.tw でお問い合わせいただけます。

[病院全体の共通サービス] 全病院共通サービスにはどのような総合サービスやネットワーク情報、書籍情報が含まれますか？

中央研究院の病院全体の共有サービスには、総合ビジネス、ネットワーク情報、書籍情報の 3 つのカテゴリが含まれます。各種サービスの具体的な内容や利用方法については、Webサイト（URL：https://its.sinica.edu.tw）でご確認ください。

[社員アカウント（SSO）の申請とパスワードの設定] スタッフ アカウント (SSO) はいつ申請できますか?

職場に到着して登録手続きを完了したら、「病院管理サービス」システムにアクセスしてスタッフアカウント（SSO）を申請できます。

[社員アカウント（SSO）の申請とパスワードの設定] パスワードの有効期限が切れる前にリマインドレターは届きますか?

はい、システムはパスワードの有効期限が切れる 7、3、または 1 日前に、パスワードを変更する必要があることを通知するリマインダー レターを送信します。

[社員アカウント（SSO）の申請とパスワードの設定] 忘れたパスワードをリセットするにはどうすればよいですか?

パスワードを忘れた場合、またはパスワードが無効な場合は、「機関管理サービス」システム (https://ssoz.apps.sinica.edu.tw/) にアクセスして従業員アカウントのパスワードをリセットできます。パスワードをリセットするときは、バインドされたバックアップ メールボックスを通じて本人確認を行う必要があります。その後、パスワードをリセットできます。

[社員アカウント（SSO）の申請とパスワードの設定] パスワードを設定する際に注意すべき原則は何ですか?

パスワード設定では、次の原則に注意する必要があります。パスワードの長さは 8 文字以上であること、大文字と小文字は別のパスワードとみなされ、少なくとも 1 つの英数字が含まれることです。過去 10 回に設定したパスワードを再使用しないでください。特殊記号を使用する場合は、一重引用符 (')、二重引用符 (")、バックスラッシュ (\\)、および空白 (スペース) を避けてください。

[社員アカウント（SSO）の申請とパスワードの設定] バインドされたバックアップ メールボックスを変更するにはどうすればよいですか?

バインドされたバックアップメールボックスを変更するには、「個人バックアップメール設定申請書」に必要事項を記入し、情報サービス窓口に直接送付して設定していただく必要があります。

[個人向けウェブサービス] Web ページのファイルをサーバーにアップロードするにはどうすればよいですか?

winscp や filezilla などの SFTP セキュリティ プロトコルをサポートするプログラムを使用して、Web ファイルをサーバーの public_html ディレクトリにアップロードできます。これにより、個人の URL を参照できるようになります。

[個人向けウェブサービス] public_html ディレクトリ内にはどのようなファイルを置くことができますか?

公開ファイルを public_html ディレクトリに配置すると、他の人があなたの個人 Web サイトでこれらのファイルを参照できるようになります。ただし、動的ページはまだサポートされていないことに注意してください。

はい、パーソナル Web サービスは毎日自動的にウイルスをスキャンします。汚染の疑いのあるファイルが見つかった場合はメールで通知するほか、Webページの利用状況データ分析レポートを自動送信する。

[個人向けウェブサービス] 汚染された疑いのあるファイルが見つかった場合、どのように通知を受けられますか?

[AWSクラウドサービスソリューション] クレジットカードを持つ学生や教育者が登録するのに推奨されるプランはどれですか?

クレジットカードをお持ちの学生および教育者には、AWS アカウントを登録するための AWS Educate プランをお勧めします。このプランでは、より多くの無料割り当てを提供でき、すべての AWS サービスを使用できるためです。また、退職や卒業しても関連する通知を受け取ることができるように、長期の個人メールボックスに登録することをお勧めします。

[AWSクラウドサービスソリューション] クレジット カードを持っていない場合は、どの AWS プランを選択すればよいですか?

クレジット カードをお持ちでない場合は、AWS スターター アカウント プランを選択して AWS アカウントを申請できます。無料割り当てが少なく、一部の AWS サービスは使用できませんが、クレジットなしで学生や教育者のニーズを満たすことができます。カード。登録には中央研究院のメールボックスを忘れずに使用してください。

[AWSクラウドサービスソリューション] AWS アカウントの登録後、無料割り当てに制限はありますか?

AWS アカウントを登録すると、無料割り当てが制限されます。 AWS アカウントを選択して AWS Educate プランに登録すると、より高い無料割り当てが得られ、すべての AWS サービスを使用できるようになります。ただし、AWS スターター アカウントを選択した場合、無料割り当ては少なくなり、一部の AWS サービスは使用できなくなります。

個人データの匿名化とは、個人データフィールドを削除、非表示、またはコードで置き換えることにより、個人データ記録で当事者を直接識別できないようにすることを指します。
