[Demande de compte d'employé (SSO) et définition du mot de passe] Quand puis-je demander un compte personnel (SSO) ?

Une fois que vous êtes arrivé au travail et que vous avez terminé les procédures d'inscription, vous pouvez vous rendre dans le système "Services administratifs de l'hôpital" pour demander un compte personnel (SSO).

[Demande de compte d'employé (SSO) et définition du mot de passe] Y aura-t-il une lettre de rappel avant l'expiration du mot de passe ?

Oui, le système vous enverra une lettre de rappel 7, 3 ou 1 jour avant l'expiration du mot de passe, vous informant que vous devez changer le mot de passe.

[Demande de compte d'employé (SSO) et définition du mot de passe] Comment réinitialiser mon mot de passe oublié ?

Si vous oubliez le mot de passe ou si le mot de passe n'est pas valide, vous pouvez accéder au système "Service administratif institutionnel" (https://ssoz.apps.sinica.edu.tw/) pour réinitialiser le mot de passe du compte de l'employé. Lors de la réinitialisation du mot de passe, vous devez vérifier votre identité via la boîte aux lettres de sauvegarde liée, puis vous pouvez réinitialiser le mot de passe.

[Demande de compte d'employé (SSO) et définition du mot de passe] Quels sont les principes auxquels faire attention lors de la définition des mots de passe ?

Les principes suivants doivent être respectés lors de la définition du mot de passe : la longueur du mot de passe doit être supérieure à 8 caractères, les majuscules et les minuscules sont considérées comme des mots de passe différents et au moins un alphanumérique est inclus. Ne réutilisez pas les mots de passe qui ont été définis les 10 dernières fois. Si des symboles spéciaux sont utilisés, évitez les guillemets simples ('), les guillemets doubles ("), les barres obliques inverses (\\) et les blancs (espace).

[Demande de compte d'employé (SSO) et définition du mot de passe] Comment changer la boîte aux lettres de sauvegarde liée ?

Pour modifier la boîte aux lettres de sauvegarde liée, vous devez remplir le "Formulaire de demande de configuration de messagerie de sauvegarde personnelle", puis l'envoyer directement au bureau du service d'information pour configuration.

[Solution de services cloud AWS] Comment Academia Sinica propose-t-elle un plan de remise sur le service cloud AWS ?

Academia Sinica a rejoint les institutions membres d'AWS Educate, afin que les collègues d'Academia Sinica puissent obtenir plus de points de quota gratuits chaque année lorsqu'ils demandent à utiliser les services cloud AWS.

[Solution de services cloud AWS] Combien de méthodes d'application existe-t-il pour le programme AWS Educate ?

Il y a deux façons de postuler.

[Solution de services cloud AWS] Quel plan est recommandé pour les étudiants et les éducateurs avec des cartes de crédit pour s'inscrire ?

Pour les étudiants et les enseignants qui ont des cartes de crédit, le plan AWS Educate pour l'enregistrement d'un compte AWS est recommandé. Parce que ce plan peut fournir un quota gratuit plus élevé et peut utiliser tous les services AWS. Et il est recommandé de s'inscrire avec une boîte aux lettres personnelle à long terme, afin de pouvoir recevoir des notifications pertinentes même si vous quittez votre emploi ou que vous êtes diplômé.

[Solution de services cloud AWS] Quel plan AWS dois-je choisir si je n'ai pas de carte de crédit ?

Si vous n'avez pas de carte de crédit, vous pouvez choisir le plan de compte AWS Starter pour demander un compte AWS Bien que le quota gratuit soit faible et que certains services AWS ne puissent pas être utilisés, il peut toujours répondre aux besoins des étudiants ou des enseignants sans crédit cartes. N'oubliez pas d'utiliser la boîte aux lettres de l'Academia Sinica pour vous inscrire.

[Solution de services cloud AWS] Une fois le compte AWS enregistré, y a-t-il une limite au quota gratuit ?

Après avoir enregistré un compte AWS, le quota gratuit sera limité. Si vous choisissez un compte AWS pour vous inscrire au plan AWS Educate, vous obtiendrez un quota gratuit plus élevé et pourrez utiliser tous les services AWS. Cependant, si vous choisissez un compte AWS Starter, le quota gratuit sera inférieur et vous ne pourrez pas utiliser certains services AWS spécifiques.
